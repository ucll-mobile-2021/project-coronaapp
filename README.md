## APIs

### COVID-19-Stats-API
Statistics screen uses the [COVID 19 Stats API](https://covid19-stats-api.herokuapp.com/) 
- Free (for everyone)
- Unlimited requests/day

### News-API
Latest news screen uses the [News API](https://newsapi.org/) 
- Free (for developers) 
- 500 requests/day
