== APIs
=== COVID-19-Stats-API
* Statistics screen uses the https://covid19-stats-api.herokuapp.com/[COVID 19 Stats API]
* Free (for everyone)
* Unlimited requests/day

=== News-API
* Latest news screen uses the https://newsapi.org/[News API]
* Free (for developers) **only for 1 month**
* 500 requests/day
